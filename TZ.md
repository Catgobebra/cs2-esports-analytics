Техническое задание на проект: Система аналитической оценки для энтузиастов ставок на киберспорт с целью определения целесообразности пари на команды Counter-Strike
1. Цель проекта: Создание веб-сервиса для аналитической оценки матчей по Counter-Strike 2 (CS2), ориентированного на энтузиастов ставок в киберспорте. Сайт позволяет пользователю выбрать две команды, получить статистику с HLTV.org, визуализацию ключевых метрик и расчёт вероятности победы каждой команды с помощью машинного обучения. Проект помогает принимать обоснованные решения о целесообразности пари, снижая зависимость от интуиции и предоставляя объективные данные и рекомендации ИИ.
2. Роли пользователей: 
**Гость**. Полный доступ к ключевому функционалу сервиса:
* Выбор двух команд из списка с поиском.
* Запуск анализа матча (кнопка «Анализ»).
* Просмотр полного отчёта без перезагрузки страницы:
* Ключевые метрики команд (рейтинг HLTV 2.0, win rate за 3 месяца, K/D, сыгранные карты, мировой ранг).
* Графики: динамика рейтинга, сравнение винрейта по картам, прогресс-бар вероятностей победы.
* Рекомендация ИИ с указанием фаворита или равного матча.
3. Модели данных:
    Модель 1: Team (Команда)
_Поля: name (уникальное название), slug, hltv_id (ID на HLTV), team_url (ссылка на страницу команды на HLTV)._
    Модель 2: DailyStats (Ежедневная статистика команды)
_Поля: team (ForeignKey → Team), parse_date (дата парсинга), rank (позиция в мировом рейтинге), maps_played (сыгранные карты), kd_diff (разница K/D), kd_ratio (K/D соотношение), rating (рейтинг HLTV 2.0), win_rate_last_3m (процент побед за последние 3 месяца)._
    Модель 3:MapStats (Статистика команды на конкретной карте)
_Поля: daily_stats (ForeignKey → DailyStats), map_name (название карты), win_rate (процент побед на карте), wins (победы), losses (поражения)._
4. Ключевой функционал:
* Пользователь выбирает две команды из выпадающих списков и нажимает кнопку «Анализ». Система динамически (через HTMX) подгружает актуальную статистику и отображает результат без перезагрузки страницы.
* Пользователь видит сравнительные карточки команд: рейтинг, процент побед за 3 месяца, K/D, количество сыгранных карт, мировой ранг.
* Пользователь получает визуализацию: линейные графики изменения рейтинга за доступный период, столбчатый график сравнения винрейта по картам, прогресс-бар с вероятностями победы.
* Пользователь читает рекомендацию ИИ: явное указание, какая команда имеет больше шансов на победу (или равный матч), на основе рассчитанных вероятностей.
5. Внешние интеграции / Аналитика: 
Машинное обучение: предобученная модель (scikit-learn), загружаемая с помощью joblib, используется в модуле cs2analit.ml.predictor для расчёта вероятности победы. Формирование признаков и предсказание выполняются с использованием pandas (создание DataFrame с признаками exp_diff, form_diff, h2h_team1).
Визуализация: Chart.js для интерактивных графиков (линейные графики рейтинга, бар-чарт по картам).
